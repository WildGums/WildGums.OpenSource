This library provides calendar and date/time functionality and extension methods for use in WildGums applications.

## Key features

- Date/time helper methods and extension methods
- Calendar utilities for working with business days, week numbers, and date ranges
- Integration with WPF date/time controls from [Orc.Controls](https://github.com/WildGums/Orc.Controls)

## Getting started

Add the `Orc.Calendar` NuGet package to your project. The library provides extension methods on `DateTime` and `DateTimeOffset` to simplify common operations such as calculating business days, determining the start and end of a period, and converting between time zones.
